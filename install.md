npm Installation Method:

1. Install Node.js (LTS version).


2. Check that Node.js and npm are installed correctly.


3. Install n8n globally using npm.


4. Create a separate user for running n8n.


5. Start n8n from the terminal.


6. Open the browser and access n8n on the default port.


7. If needed, set up n8n to run in the background using a process manager.


8. n8n will now run continuously and can be accessed anytime.


Docker Installation Method:

1. Install Docker and Docker Compose on your system.


2. Verify that Docker is running properly.


3. Create a folder for your n8n project.


4. Create a Docker Compose file inside the folder.


5. Define the n8n service, port, environment variables, and storage volume.


6. Start the n8n service using Docker Compose.


7. Confirm that the n8n container is running.


8. Open the browser and access n8n through the exposed port.


9. Stop the service anytime using Docker commands.


10. All workflow data is stored safely in the Docker volume.
